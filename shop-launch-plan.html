What this is

A complete plan to extend dayindayin.dk (Stine's portfolio site, already built on Next.js 15) into a Gelato print-on-demand store. Stine's actual artwork files in Dropbox become the print designs. Gelato handles printing and EU shipping. Shopify is the commerce layer. The dayindayin-site codebase gets a /shop section built from mikofu's proven patterns.

Architecture: Extend dayindayin-site repo — not a new repo. Add /shop, /shop/[handle], /cart pages. Shop and portfolio live on the same domain.

No Fluid adaptation layer. dayindayin is a straight artist store — Claude generates copy once at seed time, no runtime AI.
